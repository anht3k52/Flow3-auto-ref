## 🚀 Cài đặt

1. Sao chép kho lưu trữ:
   ``` bash
   git clone https://github.com/anht3k52/Flow3-auto-ref.git
   cd Flow3-auto-ref
   ```
2. Cài đặt phụ thuộc:
   ``` bash
   npm install
   ```
3. Tạo tệp codeflow3.txt và thêm mã giới thiệu của bạn :
   ``` bash
   nano codeflow3.txt
   ```
4. Tạo tệp proxies.txt và thêm proxy của bạn :
   ``` bash
   nano proxies.txt
   ```
5. Chạy Bot
   ``` bash
   npm start
   ```
